# Docker Catalog
docker-catalog は、様々なツールや開発環境を構築するためのDockerファイルを管理するリポジトリです。
ここでは、一般的なツールや技術スタックに対応するDocker環境を提供しています。

# リポジトリ構成

```text
/docker-catalog
├── tool-A/
│   ├── Dockerfile
│   ├── .dockerignore
│   └── README.md
├── tool-B/
│   ├── docker-compose.yml
│   ├── .dockerignore
│   └── README.md
└── README.md
```
